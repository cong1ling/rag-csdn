## 前置要求

- Java 17+
- Maven 3.6+
- MySQL 8.0+
- OpenAI 兼容聊天模型服务
- 阿里云 DashScope
- 阿里云 DashVector

## 配置步骤

### 1. 数据库配置

修改 `src/main/resources/application.yml`：

```yaml
spring:
  datasource:
    url: jdbc:mysql://127.0.0.1:3306/rag_csdn?createDatabaseIfNotExist=true&useUnicode=true&characterEncoding=utf8&useSSL=false&allowPublicKeyRetrieval=true&serverTimezone=Asia/Shanghai
    username: ${DB_USERNAME:root}
    password: ${DB_PASSWORD:123456}
```

说明：

- 当前默认数据库名为 `rag_csdn`
- Flyway 会自动执行数据库迁移；已有旧库会被迁移到 `article/article_id/source_id` 结构

### 2. OpenAI 兼容模型配置

PowerShell：

```powershell
$env:OPENAI_API_KEY="your_openai_api_key"
$env:OPENAI_BASE_URL="https://your-openai-compatible-endpoint"
```

如需切换聊天模型，修改：

```yaml
spring:
  ai:
    openai:
      chat:
        options:
          model: qwen3.6-plus
```

### 3. DashScope Embedding 配置

PowerShell：

```powershell
$env:DASHSCOPE_API_KEY="your_dashscope_api_key"
```

默认 Embedding 配置：

```yaml
spring:
  ai:
    dashscope:
      embedding:
        options:
          model: text-embedding-v4
          dimensions: 1024
```

### 4. DashVector 配置

PowerShell：

```powershell
$env:DASHVECTOR_API_KEY="your_dashvector_api_key"
$env:DASHVECTOR_ENDPOINT="your_dashvector_endpoint"
```

默认配置：

```yaml
spring:
  ai:
    alibaba:
      dashvector:
        collection: csdn
```

创建 DashVector Collection 时建议：

- Collection 名称：`csdn`
- 向量维度：`1024`
- 距离度量：`cosine`

## 启动项目

### Maven 启动

```powershell
cd rag-csdn-server
mvn spring-boot:run
```

### IDEA 启动

运行主类：

```text
com.example.ragcsdn.RagCsdnApplication
```

### 打包运行

```powershell
mvn clean package -DskipTests
java -jar target/rag-csdn-server-1.0.0.jar
```

## 常见问题

### 1. 数据库连接失败

- 检查 MySQL 是否启动
- 检查用户名、密码和端口
- 检查 `rag_csdn` 是否可访问

### 2. DashVector 连接失败

- 检查 `DASHVECTOR_API_KEY`
- 检查 `DASHVECTOR_ENDPOINT`
- 检查 Collection `csdn` 是否已创建

### 3. 文章导入失败

- 检查传入 URL 是否为可访问的 CSDN 文章地址
- 检查目标文章正文是否可被提取
- 查看后端日志中的失败原因回写
